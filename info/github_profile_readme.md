## 👨‍💻 Hakkımda (About Me)

Ben **Kaan**, **PromSoftware** bünyesinde masaüstü, web ve sistem otomasyon çözümleri geliştiren tutkulu bir yazılım geliştiriciyim. C#, .NET ve JavaScript/TypeScript ekosisteminde çalışarak kullanıcı deneyimini ön planda tutan, yerel odaklı (local-first) ve yüksek performanslı uygulamalar üretiyorum.

* 🏢 **PromSoftware** bünyesinde Kurucu & Geliştirici olarak projeler üretiyorum.
* 🚀 Masaüstü (Electron, .NET WPF/WinForms) ve Web teknolojileri üzerinde uzmanlaşıyorum.
* 🔒 Veri gizliliği, yerel kriptografik veri depolama (AES-256) ve ağ tünelleme sistemleriyle ilgileniyorum.
